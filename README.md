# AgroGuardião 360

## Agro forte, futuro sustentável

O **AgroGuardião 360** é um projeto desenvolvido para o **Agrinho 2026**, na categoria de Programação Front-End, com o tema **“Agro forte, futuro sustentável”**.

O objetivo do projeto é mostrar, de forma interativa, como a tecnologia pode ajudar o campo a produzir mais, economizar recursos naturais e proteger o meio ambiente.

## Objetivo do projeto

O projeto apresenta uma simulação de monitoramento da lavoura, onde o usuário pode alterar dados como:

* Umidade do solo
* Temperatura
* Sinais nas folhas
* Preservação ambiental

Com essas informações, o sistema calcula um nível de risco da lavoura e apresenta recomendações sustentáveis para o produtor rural.

## Funcionalidades

O site possui:

* Página inicial com apresentação do projeto
* Simulador de risco da lavoura
* Botão para reiniciar a análise
* Botão para salvar análise no histórico
* Painel com dados atualizados
* Quiz educativo sobre agro sustentável
* Gerador de relatório final
* Botão de acessibilidade para aumentar a fonte
* Alternância de tema visual

## Tecnologias utilizadas

O projeto foi desenvolvido com:

* HTML5
* CSS3
* JavaScript

Não foram utilizados frameworks ou bibliotecas externas. O projeto foi construído com arquivos separados, seguindo a estrutura básica de desenvolvimento Front-End.

## Como usar

1. Acesse a página inicial do projeto.
2. Leia a proposta sobre tecnologia e sustentabilidade no campo.
3. Vá até o simulador.
4. Ajuste os valores de umidade, temperatura, sinais nas folhas e preservação ambiental.
5. Observe o resultado da análise.
6. Salve a análise no histórico, se desejar.
7. Responda ao quiz para testar seus conhecimentos.
8. Gere um relatório final com uma ação sustentável.

## Tema abordado

O projeto trabalha a relação entre campo, tecnologia e sustentabilidade. A proposta mostra que o uso de dados, sensores e monitoramento pode ajudar o produtor rural a tomar decisões mais conscientes, reduzindo desperdícios e preservando recursos naturais.

## Importância do projeto

A agricultura é essencial para a sociedade, mas precisa caminhar junto com a preservação ambiental. O **AgroGuardião 360** mostra que a tecnologia pode ser uma aliada para construir um futuro mais equilibrado, produtivo e sustentável.
